Wildfires Affect on East Troublesome Watershed

2 Methods

Downloaded data from USGS site number 09034250 that is along the Colorado River that was affected by the East Troublesome Fire.

2.1 Downloaded Discharge Data

site_no <- "09034250" 

# Discharge data pulled form the site from 10/01/1986 to 1/1/2022
q <- readNWISdv(siteNumbers = site_no,
                parameterCd = '00060',
                startDate = '1986-10-01',
                endDate = '2022-01-01') %>%
  rename(q_cfs = X_00060_00003) %>%
  addWaterYear(.) %>%
  select(-agency_cd, -X_00060_00003_cd)

2.1.1 Mean Monthly Discharge

q_month_means <- q%>%
  mutate(month = month(Date),
         year = year(Date))%>%
  group_by(year, month)%>%
  summarize(q_mean = mean(q_cfs, na.rm=T))
